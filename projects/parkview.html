About the Project

Developed for the Parkview Packnett Family Cancer Institute, this application addresses a critical need in healthcare data management. The Precision Genomics team at Parkview Health processes numerous genomic sequencing test results that are currently stored as PDF documents in patient medical records.

The program automates the extraction of patient data from these PDFs, converts it into structured data fields, and stores it in a secure database. This enables healthcare providers to:

  • Quickly locate and access patient genomic data
  • Run automated reports on the structured data
  • Reduce manual data entry and potential errors
  • Efficiently correlate genomic sequences with patient information

Due to the sensitive nature of healthcare data and HIPAA compliance requirements, the source code is available upon request. Please contact me for more information.
